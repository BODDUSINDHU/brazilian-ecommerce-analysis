# 📊 Brazilian E-Commerce Analytics Dashboard (Excel)

## 📌 Overview

Built an end-to-end business analytics dashboard using Brazilian Olist e-commerce data to analyze revenue performance, customer behavior, delivery efficiency, and customer satisfaction.

---

## 🎯 Business Problem

The data was spread across multiple disconnected tables (orders, customers, products, payments, reviews), making it difficult for stakeholders to:

* Track revenue trends
* Monitor delivery performance
* Understand customer behavior
* Identify growth opportunities

---

## 🛠️ Tools Used

* Microsoft Excel
* Power Query (Data Cleaning & Transformation)
* Pivot Tables & Charts
* Dashboard Design

---

## 🔍 Approach

* Merged multiple datasets into a **single analytical table**
* Defined key business KPIs:

  * Revenue, Orders, AOV
  * Delayed Delivery %
  * Repeat Customer %
  * Average Review Score
* Built pivot-based analysis layer
* Designed an **interactive dashboard with slicers**

---

## 📈 Key Insights

* 💰 Revenue: R$15.8M from 98K+ orders
* 🚚 Delayed Deliveries: Only 6.59% (strong operations)
* ⭐ Avg Review Score: 4.03 (positive customer experience)
* 🔁 Repeat Customers: 25.83% → retention improvement needed
* 🛍️ Large & Medium products drive most revenue
* 💳 Card payments dominate → digital adoption is high

---

## 📊 Dashboard Features

* KPI summary (Revenue, Orders, AOV, Delivery, Reviews)
* Monthly sales trends
* Product performance analysis
* Delivery performance tracking
* Customer satisfaction insights
* Payment method distribution
* Regional sales analysis

---

## 💡 Business Impact

* Identified **low customer retention** as key growth gap
* Highlighted **revenue concentration in few regions**
* Enabled **data-driven decision-making** for stakeholders
